## Summary

Artifact Atlas is a project asset graph, Artifact BOM, and context-pack builder for the Agentic OS. It gives humans and agents a shared view of project materials, expected artifacts, missing context, provenance, policy, and handoff bundles.

## MVP Scope

- Project home and asset library.
- Asset detail and metadata editing.
- Inbox/capture triage.
- Artifact BOM overview and seed template application.
- Manual slot assignment and coverage/gap calculation.
- Context pack builder and YAML manifest export.
- Read-first CLI/MCP agent gateway.
- MeatyWiki asset card export.

## First Release Acceptance

- Create a project and import assets.
- Browse assets in gallery/table.
- Link an asset to an IntentTree node manually.
- Apply at least one template to create BOM slots.
- Assign an asset to a BOM slot manually.
- View BOM coverage and missing slots.
- Create a context pack from selected assets or a node.
- Export asset cards/context packs to Markdown/YAML.
